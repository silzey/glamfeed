# **App Name**: GlamFeed

## Core Features:

- Cosmetic Product Reviews Feed: Display a feed of cosmetic product reviews from various users.
- User Review Submission: Allow users to submit reviews for cosmetic products, including text and images.
- AI-Powered Review Summarization: Use a tool to generate brief summaries of longer reviews for quick browsing.
- Product Search and Filtering: Enable users to search for specific cosmetic products and filter reviews based on criteria like skin type, rating, etc.
- User Profile Management: Allow users to create and manage their profiles, including saving favorite products and reviews.
- Like and Comment Functionality: Enable users to like and comment on reviews to foster interaction.

## Style Guidelines:

- Primary color: Light orchid (#E6B8B0) to evoke femininity and sophistication, referencing classic cosmetics design without being dated.
- Background color: Soft beige (#F2E9E4), offering a gentle contrast that keeps the visual focus on the products and reviews.
- Accent color: Deep rose (#C78283) to draw attention to key interactive elements.
- Body and headline font: 'Belleza', a humanist sans-serif suited to art, fashion, and design
- Use elegant, minimalist icons for navigation and user actions.
- Design a clean, card-based layout similar to Instagram to display reviews effectively.
- Incorporate subtle animations for transitions and feedback to improve user experience.
